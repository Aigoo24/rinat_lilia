# Сайт-приглашение на свадьбу Рината и Лилии

Адаптивный одностраничный сайт-приглашение под телефон и компьютер. Внутри уже стоят имена, дата, программа дня, адреса, дресс-код и фотографии.

## Что внутри

- `index.html` — структура сайта.
- `styles.css` — дизайн и адаптив.
- `script.js` — дата для таймера и отправка формы.
- `google-apps-script.gs` — код для записи ответов гостей в Google Таблицу.
- `assets/` — фотографии сайта.
- `index-standalone.html` — версия с CSS и JS внутри одного HTML-файла, но фотографии всё равно берутся из папки `assets/`.
