# Сайт-приглашение на свадьбу Рината и Лилии

Адаптивный одностраничный сайт-приглашение под телефон и компьютер. Внутри уже стоят имена, дата, программа дня, адреса, дресс-код и фотографии.

## Что внутри

- `index.html` — структура сайта.
- `styles.css` — дизайн и адаптив.
- `script.js` — дата для таймера и отправка формы.
- `google-apps-script.gs` — код для записи ответов гостей в Google Таблицу.
- `assets/` — фотографии сайта.
- `index-standalone.html` — версия с CSS и JS внутри одного HTML-файла, но фотографии всё равно берутся из папки `assets/`.

## Как подключить Google Таблицу

1. Создайте новую Google Таблицу.
2. Откройте `Расширения → Apps Script`.
3. Удалите стартовый код и вставьте код из файла `google-apps-script.gs`.
4. Нажмите `Развернуть → Новое развертывание`.
5. Тип выберите `Веб-приложение`.
6. В поле `Выполнять от имени` выберите `Я`.
7. В поле доступа выберите `Все` / `Anyone`.
8. Скопируйте ссылку Web App.
9. Откройте `script.js` и вставьте ссылку сюда:

```js
const SITE_CONFIG = {
  googleScriptUrl: 'сюда вставить ссылку'
};
```

После этого ответы из формы будут попадать на лист `Ответы гостей`.

## Как открыть сайт

Откройте файл `index.html` в браузере. Для публикации можно использовать обычный хостинг, Netlify, Vercel или GitHub Pages.

## Как заменить фото

Фотографии лежат в папке `assets/`:

- `hero.jpg` — фото в первом блоке под именами и датой.
- `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg` — фото в галерее.
- `final.jpg` — фото в финальном блоке «До встречи».

Чтобы заменить фотографию, просто положите новую с таким же названием в папку `assets/`.
